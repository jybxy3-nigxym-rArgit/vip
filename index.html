الأرقام المميزة

اكتشف عالم الأرقام المميزة والتي تسمى أيضاً أرقام ناريسيها أو أرقام أرمسترونغ

ما هي الأرقام المميزة؟

الرقم المميز (أو رقم أرمسترونغ) هو رقم يساوي مجموع أرقامه مرفوعة إلى قوة عدد أرقامه.

مثال: الرقم 153 هو رقم مميز لأن:

1³ + 5³ + 3³ = 1 + 125 + 27 = 153

حيث أن الرقم 153 يتكون من 3 أرقام، لذلك نقوم برفع كل رقم إلى أس 3.

أمثلة على الأرقام المميزة

الرقم 371

3³ + 7³ + 1³ = 27 + 343 + 1 = 371

الرقم 407

4³ + 0³ + 7³ = 64 + 0 + 343 = 407

الرقم 1634

1⁴ + 6⁴ + 3⁴ + 4⁴ = 1 + 1296 + 81 + 256 = 1634

تحقق من الرقم

أدخل رقماً للتحقق مما إذا كان رقمًا مميزًا
